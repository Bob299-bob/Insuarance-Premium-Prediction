# Insurance Premium Prediction 💰

An AI-powered Machine Learning web application that predicts medical insurance premium charges based on user details like age, BMI, smoking habits, gender, number of children, and region.

## 🚀 Features

* Predict insurance charges instantly
* Interactive and user-friendly UI
* Machine Learning based prediction system
* Real-time result generation
* Responsive Streamlit web app
* Data preprocessing and feature encoding

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Joblib

## 📂 Project Workflow

1. Data Collection & Analysis
2. Data Cleaning & Preprocessing
3. Feature Encoding
4. Model Training
5. Model Saving using Joblib
6. Streamlit Web App Deployment

## 📊 Input Parameters

* Age
* Gender
* BMI
* Number of Children
* Smoking Status
* Region

## 🎯 Output

The application predicts the estimated insurance premium/medical charges for the user.
